# Landing Page

A landing page project built as part of The Odin Project Foundations curriculum. This project focuses on applying HTML, CSS, and Flexbox skills to recreate a provided design layout.

## Project Overview

The goal of this project is to build a complete landing page based on a design mockup. The page consists of:

* Hero section
* Information section
* Quote section
* Call-to-action section
* Footer

The project emphasizes structuring content with semantic HTML and styling layouts using Flexbox.

## Features

* Clean and organized HTML structure
* Flexbox-based layout
* Custom colors and typography
* Multiple page sections matching the provided design
* Responsive principles considered where applicable

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Git & GitHub

## What I Learned

Through this project, I practiced:

* Building a webpage from a design mockup
* Creating layouts with Flexbox
* Organizing content into reusable sections
* Styling buttons, cards, and text elements
* Managing a project using Git commits
* Pushing and maintaining code on GitHub
